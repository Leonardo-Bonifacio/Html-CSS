Parametro href para colocar a URL do lnk, parámetro target para
escolher se o link abrira uma nova pagina ou ira sobrepor o site
atual(_blank ou _self), Parámetro rel para dizer para onde esse link levara
(external,prev,next,nofollow e author), parámetro hreflang para escolher a
linguagem principal do site que o link levará
